In this lab, you will build a "Startup Name Generator." Users will type an industry into a text box, click a button, and see a list of 10 creative company names generated by AI. 

We will walk through how to build this using the different tools we covered in Module 2.

<SectionLabel label="Option A: Building with Firebase Studio" />

Firebase Studio handles the frontend, backend, and deployment in one place.

1.  **Setup:** Go to Firebase Studio and create a new project.
2.  **The Initial Prompt:** Paste your PRD into the main chat box.
    *"Create a startup name generator. It needs a text input field for 'industry' and a 'Generate' button. When the button is clicked, generate a list of 10 creative names and display them below. Use a dark theme with neon green accents."*
3.  **Review the Blueprint:** Firebase Studio will stop and show you a "Blueprint" of its plan. Review it. If you want a "Copy to Clipboard" button next to each name, tell the AI to add it now, before it writes code.
4.  **Generate:** Approve the blueprint. The AI will write the code and show you a live preview.
5.  **Publish:** Click the "Deploy" button. Your app is now live on the internet.

<SectionLabel label="Option B: Building with Lovable.dev" />

Lovable is highly visual and excellent for front-end design.

1.  **Setup:** Go to Lovable.dev and start a new project.
2.  **Design the UI First:** Ask the AI to build the visual interface before adding logic.
    *"Build a dark-themed page. Center a title that says 'Name Generator'. Below it, add a wide text input field and a purple gradient button that says 'Generate'. Below that, create a grid layout to hold 10 cards."*
3.  **Add the Logic:** Once the design looks good, add the AI functionality.
    *"When I click the Generate button, take the text from the input field, use an AI API to generate 10 startup names based on that text, and display one name inside each card in the grid below."*
4.  **Iterate:** If the names look boring, refine the prompt: *"Tell the AI generating the names to act like a high-end branding agency."*
5.  **Publish:** Use Lovable's built-in deploy feature to put the site online.

<SectionLabel label="Option C: Building with Replit" />

Replit gives you full access to the code and backend services.

1.  **Setup:** Go to Replit and create a new "Next.js" template project.
2.  **The Initial Prompt:** Ask the Replit AI to generate the core files.
    *"Create a Next.js application for a name generator. I need a frontend page with a text input and a submit button. I also need a backend API route that takes the industry text, calls an LLM to generate 10 names, and returns them as a JSON list."*
3.  **Debug:** If the code fails to run, you will see a red error message in the console. Copy the entire error message, paste it into the chat, and say: *"I received this error. Fix it."*
4.  **Publish:** Click the "Deploy" button in the top right corner of the Replit interface.
